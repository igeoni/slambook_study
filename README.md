# slambook_study

This repository contains personal study code following the tutorials from  
*slambook2* by Gaoxiang Gao.

The implementations are based on the official reference repository:  
[gaoxiang12/slambook2](https://github.com/gaoxiang12/slambook2)

This project focuses on understanding core SLAM concepts using C++, CMake, Eigen, and GTSAM.
